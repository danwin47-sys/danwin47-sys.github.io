## 6. How to Work with This Project (For AI Agents)

### When Making Updates

1. **Content Changes**:
   - Update `index.html` for text/structure
   - Maintain professional tone
   - Keep vibe coding philosophy prominent

2. **Design Changes**:
   - Use CSS Variables for consistency
   - Test dark mode compatibility
   - Ensure responsive behavior

3. **Feature Additions**:
   - Add to `js/main.js`
   - Follow existing code patterns
   - Test across devices

4. **Deployment**:
   - Commit changes to main branch
   - Push to GitHub
   - Verify GitHub Actions runs successfully
   - Check live site after 1-2 minutes

### Best Practices

1. **Always test responsive design** (mobile, tablet, desktop)
2. **Verify dark mode** works correctly
3. **Check all links** and navigation
4. **Maintain accessibility** (ARIA labels, semantic HTML)
5. **Optimize performance** (minimize CSS/JS, optimize images)
6. **Keep documentation updated** (README, DEPLOYMENT)

### Common Tasks

- **Update Projects**: Edit project cards in `index.html`
- **Change Colors**: Modify CSS Variables in `:root`
- **Add Skills**: Update skills section HTML
- **Update Contact**: Change email/social links
- **Add Features**: Extend `js/main.js` with new functionality
